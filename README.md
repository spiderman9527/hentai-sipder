### 绅士资源爬虫

#### 1.创建虚拟环境
创建虚拟环境并开启
```bash
python -m venv [venv-folder] & [venv-folder]/Scripts/activate
```
关闭虚拟环境：
```bash
[venv-folder]/Scripts/deactivate
```
> 该步骤是为了隔离环境，如果你使用全局安装可跳过该步骤，请先确保系统已安装了python

#### 2. 安装依赖
```shell
pip install -r requirements.txt
```
#### 3. 启动程序

```bash
python src/main.py
```
